# Python GUI Keylogger Demonstration (Cybersecurity Internship Project)

This repository contains a **Python-based educational keylogger demonstration**, built to showcase how monitoring tools work in cybersecurity.  
It includes a **Tkinter GUI Control Panel** that allows users to Start and Stop logging ethically.

> This project is strictly for ethical learning and must **only** be run on your own machine.

---

##  Features
- GUI (Start / Stop / Exit Controller)
- Logs every keystroke individually
- Saves keystrokes with timestamps
- Uses threading for smooth GUI
- Good for cybersecurity skill demonstration
